⚠️ Troubleshooting

I see "Not authenticated" error

You must be logged in as an admin user. Contact your database administrator to add the admin flag to your user account in Firebase.

Collections already exist error

The collections have already been created. This is normal - you can proceed to the next step. If you need to reset, manually delete collections in the Firebase console.

Can't join competitions

Make sure you've completed all steps. Users need to be logged in and have access to the comps.html page.
