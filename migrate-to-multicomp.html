🔄 Migrate to Multi-Comp System

This migration tool will move your existing competition data from the old system to the new multi-comp structure.

Step 1: Create Default Competition

This creates the "The Great Spring Tip Off" competition document in the new system.

Step 2: Update Races with Competition ID

This adds the compId field to all existing races so they're associated with the competition.

Step 3: Migrate Users to Competition Joinings

This creates joining records for all users who have made tips, so they can access the competition.

Step 4: Migrate Leaderboard Points

This migrates the actual points data from the old leaderboard collection to userCompJoinings for each user.

Step 5: Verify Migration

Check that the migration was successful.

⚠️ Important

  • ✅ This migration is safe - it doesn't delete any data
  • ✅ You can run this multiple times without issues
  • ✅ After migration, users need to be in userCompJoinings to join comps
  • ✅ Once complete, comps will be visible in comps.html
  • ⚠️ Make sure you have admin access before proceeding
